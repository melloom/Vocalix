## ðŸ“– General Terms

### Clip
A voice recording shared on Echo Garden. Can be 30 seconds (regular) or up to 10 minutes (podcast mode).

### Profile
A user account on Echo Garden. Identified by a handle and emoji avatar.

### Handle
A user's display name/username on Echo Garden. Can be changed (with limits).

### Topic
A daily discussion prompt that inspires voice clips. New topics are generated automatically each day.

### Community
A group centered around a topic, interest, or theme. Similar to subreddits but audio-first.

### Live Room
A real-time audio discussion room. Similar to Clubhouse or Twitter Spaces.

## ðŸ”§ Technical Terms

### RLS (Row Level Security)
Database-level security feature that restricts access to rows based on policies. Used throughout Echo Garden.

### Edge Function
Serverless function running on Supabase. Used for background processing like transcription and moderation.

### PostgREST
Auto-generated REST API from PostgreSQL database schema. Provides automatic API endpoints.

### Realtime
WebSocket-based real-time updates. Allows live updates when data changes in the database.

### Device ID
Unique identifier stored in browser localStorage. Used for device-based authentication.

### Magic Link
Passwordless authentication method. User receives email with login link.

### Supabase
Backend-as-a-Service platform providing database, storage, authentication, and edge functions.

## ðŸŽ¯ Feature Terms

### Voice Reaction
A short 3-5 second voice clip used as a reaction to another clip.

### Chain
A series of linked clips that tell a story or continue a conversation.

### Remix
A clip that adds voice overlay to another clip, creating an audio mashup.

### Trending Score
Algorithm-calculated score that determines how "trending" a clip is. Based on engagement, freshness, and quality.

### Quality Badge
Badge awarded based on audio quality (excellent, good, fair). Automatically calculated.

### Emotion Detection
AI-powered detection of emotions in voice clips (joy, sadness, anger, etc.).

### Voice Characteristics
Analysis of voice features like pitch, speed, tone, and timbre.

### Offline Mode
Ability to download clips for offline listening. Stored in browser IndexedDB.

### Background Playback
Ability to continue playing audio when app is in background or minimized.

### Daily Digest
Email digest containing best clips from topics you follow. Can be daily or weekly.
