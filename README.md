# Echo Garden

Echo Garden is a Vite + React application for capturing and sharing short voice notes. The project ships with a Supabase backend, shadcn-ui components, and a polished mobile-first layout.

## Getting Started

```sh
# Clone the repository
git clone <YOUR_GIT_URL>
cd echo-garden-49-main

# Install dependencies
npm install

# Start the Vite dev server
npm run dev
```

The development server runs on `http://localhost:8080/` by default.

## Tech Stack

- Vite + TypeScript + React
- Tailwind CSS and shadcn-ui
- Supabase JavaScript client
- TanStack Query for data fetching

## Deployment

When you're ready to deploy, build the project with:

```sh
npm run build
```

You can deploy the contents of the `dist` directory to any static hosting provider such as Netlify or Vercel.
